Portfolio site for Los Angeles based art director Josh Kopeika. Josh Kopeika’s site showcases his work for Nike, Adidas and New Balance to name a few. The site uses a left to right image layout for visual variety and from a UX perspective elements animate gracefully up from the bottom of the screen. Josh’s work is presented in one long scroll, as we believed that clicking through to project negate user experience, and having it in this simply long feed format would maximise retention.
